Execute from the command line.

Compiled on Ubuntu 14.04 32bit with g++ 4.8.2

Stable Matching:
$ chmod +x stable-matching
$ ./stable-matching ../../input/stable-matching/input.txt

String Matching:
$ chmod +x string-matching
$ ./string-matching ../../input/string-matching/input.txt

Simplex:
$ chmod +x simplex
$ ./simplex ../../input/simplex/input.txt

Max-flow:
$ chmod +x max-flow
$ ./max-flow ../../input/max-flow/input.txt
